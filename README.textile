h1. Sample OAuth2 Server Application using Rack::OAuth2 in Rails3

h2. Setup & Run

<pre>
git clone git@github.com:nov/rack-oauth2-sample.git
cd rack-oauth2-sample
bundle install
rake db:migrate
rails server
</pre>

Then go to http://localhost:3000
